# ChunkForge

## 概述
ChunkForge 是一个分布式文件存储教学演示平台。上传任意文件后，系统使用 Reed-Solomon (4+2) 纠删码将其切分为 6 个分片，均匀分布至不同虚拟节点。你可以在界面中动态增删节点，点击“宕机”模拟故障，系统将自动触发数据修复并在前端实时展示进度。只要仍有任意 4 个可用分片，即可完整还原文件。

## 核心功能
- **集群管理**：动态添加/删除虚拟存储节点，查看心跳与状态。
- **分片引擎**：k=4, m=2 纠删码自动切分与重组。
- **故障模拟**：手动标记节点宕机，观察自动修复流程。
- **可视化仪表盘**：节点拓扑、分片分布热力图及操作日志。

## 技术栈
- 后端：Python FastAPI
- 前端：React, D3.js
- 存储：文件系统模拟（每个节点一个目录）
- 部署：Docker Compose

## 快速开始
1. 启动集群：`docker-compose up -d`
2. 打开浏览器访问 `http://localhost:3000`
3. 按照 `docs/` 中的节点通信协议和纠删码规范理解内部设计。

## 测试
涵盖并发上传下载、随机宕机数据恢复、脑裂防护等自动化测试，执行 `pytest` 验证分布式逻辑正确性。
