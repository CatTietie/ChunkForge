# 分布式文件存储节点模拟器 - 构建计划

## 目标
构建一个教学和演示用的分布式存储集群模拟系统，可以直观地看到文件分片如何分布在不同节点上，并模拟节点故障后的自动修复过程。

## 核心模块
- **集群节点管理器**：运行多个虚拟存储节点（进程内模拟），维护心跳与元数据
- **文件分片引擎**：实现固定大小分块 + 纠删码冗余，分配分片到不同节点
- **上传下载代理**：接收用户文件，切分分发，读取时重组
- **可视化仪表盘**：节点拓扑图、分片分布热力图、集群健康度
- **故障注入与修复**：手动“kill”节点，观察数据自动重新分布

## 分步执行阶段
1. **设计通信协议**：节点间心跳、元数据同步、分片读写接口
2. **实现单节点存储与心跳**：每个节点维护本地文件碎片
3. **分片策略与分配算法**：实现4+2纠删码，节点间均匀分布
4. **文件上传/下载全流程**：处理并发读写
5. **故障检测与修复**：心跳超时判定，触发数据重分布
6. **前端可视化**：节点图、分片分布、操作日志
7. **异常场景全覆盖测试**：脑裂、并发故障、数据一致性校验
